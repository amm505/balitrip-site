# AGENTS

## Project
Balitrip production website.

## Rules
- Do not edit production directly.
- All changes go through GitHub branches and pull requests.
- Main branch is production.
- Develop branch is staging/integration.
- Keep changes small and reviewable.
- Do not change deployment or infrastructure without explicit approval.
- Do not store secrets in the repository.

## Structure
- site/ — website pages
- assets/ — images and media
- css/ — styles
- js/ — scripts
- docs/ — project documentation
- infra/ — infrastructure files
- .github/workflows/ — GitHub automation
