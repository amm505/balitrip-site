# AI Development Rules

## Workflow
AI agents must:

1 Create feature branch from develop
2 Make changes only in feature branch
3 Create pull request to develop
4 Never push directly to main
5 Never push directly to develop

## Code rules

Keep changes small.

Do not modify infrastructure without approval.

Do not change deployment without approval.

Do not delete files without explanation.

## Review rules

All changes must go through PR.

Human approval required.
